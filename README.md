# 📝 Task Manager

Um aplicativo de gerenciamento de tarefas desenvolvido com React.

O objetivo do projeto é praticar conceitos fundamentais do React como componentização, hooks, manipulação de estado e persistência de dados no navegador.

---

## 🚀 Funcionalidades

- Criar novas tarefas
- Editar tarefas existentes
- Excluir tarefas
- Persistência de dados com LocalStorage
- Interface simples e responsiva

---

## 🛠 Tecnologias utilizadas

- React
- JavaScript (ES6+)
- CSS (ou Tailwind, caso esteja usando)
- Vite
- LocalStorage API

# Task Manager — React

A task management application built with React, focused on productivity and daily organization.

This project allows users to create, edit, delete, and manage tasks with data persistence in the browser.

---

## Features

* Create, edit, and delete tasks
* Mark tasks as completed
* Filter tasks by status (all, pending, completed)
* Data persistence using LocalStorage
* Responsive and intuitive interface
* Dynamic task list updates

---

## Concepts applied

* React component-based architecture
* Hooks (useState, useEffect)
* State management
* Frontend data handling
* Browser data persistence
* SPA structure fundamentals

---

## Technologies used

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite
* LocalStorage API

## How to run locally

### 1. Clone the repository

```bash id="qv07ld"
git clone https://github.com/rodrigovalentimrs/task-manager.git
```

### 2. Navigate to the project folder

```bash id="x5o3df"
cd task-manager
```

### 3. Install dependencies

```bash id="5z6h0n"
npm install
```

### 4. Run the project

```bash id="8c2k3a"
npm run dev
```
